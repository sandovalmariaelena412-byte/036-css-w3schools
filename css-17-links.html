CSS Styling Links

Los enlaces se pueden estilizar con cualquier propiedad CSS (color, font-family, background, etc.). Ademas, se pueden estilizar de forma distinta dependiendo del estado en que se encuentren.

Los 4 estados de un enlace

a:link (enlace normal, no visitado), a:visited (enlace visitado), a:hover (cuando el mouse pasa por encima), a:active (el momento en que se hace clic).

Enlace con color segun su estado

Reglas de orden: a:hover debe ir despues de a:link y a:visited. a:active debe ir despues de a:hover.
