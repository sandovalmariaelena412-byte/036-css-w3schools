Demonstrating the Box Model

The CSS box model is essentially a box that wraps around every HTML element.

It consists of (from innermost part to outermost part): content, padding, border, and margin.

Calculate the total width and height

This div element will have a total width of 350px and a total height of 80px.

Let's do the math

320px (width of content area) + 20px (left padding + right padding) + 10px (left border + right border) + 0px (left margin + right margin) = 350px (total width)

50px (height of content area) + 20px (top padding + bottom padding) + 10px (top border + bottom border) + 0px (top margin + bottom margin) = 80px (total height)

Formulas

Total element width = width + left padding + right padding + left border + right border + left margin + right margin

Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

Nota: la propiedad margin tambien afecta el espacio total que ocupa la caja en la pagina, pero el margin no se incluye en el tamano real de la caja (el ancho y alto total terminan en el borde).
