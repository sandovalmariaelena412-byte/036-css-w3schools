An Image as The List Item Marker

La propiedad list-style-image especifica una imagen como marcador del elemento de lista. Nota: siempre se debe especificar tambien list-style-type, por si la imagen no esta disponible.

  • Tea

Position The List Item Markers

La propiedad list-style-position especifica la posicion de los marcadores.

outside (por defecto): el marcador queda fuera del elemento de la lista

  • Coffee - Este texto es una descripcion muy larga que ocupara mas de una linea. Se puede ver que el marcador esta fuera de la lista y la segunda linea del texto no queda alineada con el marcador.
  • Tea
  • Coca Cola

inside: el marcador queda dentro del elemento de la lista

  • Coffee - Este texto es una descripcion muy larga que ocupara mas de una linea. Se puede ver que el marcador esta dentro de la lista y forma parte del texto, empujandolo desde el inicio.
  • Tea
  • Coca Cola

Remove Default Settings

La propiedad list-style-type:none; se usa para eliminar los marcadores de los elementos de lista. Nota: una lista tiene margen y padding por defecto, para eliminarlos se agrega margin:0 y padding:0 al ul o ol.

  • Coffee
  • Tea
  • Coca Cola

List - Shorthand property

La propiedad list-style es una forma abreviada. El orden de los valores es: list-style-type, list-style-position, list-style-image.
