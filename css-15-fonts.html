Font Family

La propiedad font-family debe contener varios nombres de fuente como sistema de "respaldo" (fallback). Si el navegador no soporta la primera fuente, prueba con la siguiente. Siempre se debe terminar con una familia generica.

Font Web Safe

Las fuentes web safe son fuentes que estan disponibles en casi todos los sistemas y navegadores. Arial es una de las fuentes web mas seguras, esta disponible en casi todos los sistemas operativos.

Font Fallbacks

Siempre es importante usar fuentes de respaldo: una lista de fuentes similares en la propiedad font-family. Si la primera no funciona, el navegador prueba la siguiente, y asi sucesivamente.

Este texto usa Tahoma, con Verdana y sans-serif como respaldo.
