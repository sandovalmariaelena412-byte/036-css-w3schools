CSS Outline

La propiedad outline agrega una linea alrededor de un elemento, fuera del borde, para hacerlo "destacar". Nota: el outline no es parte de las dimensiones del elemento; el ancho y alto total del elemento no se ven afectados por el ancho del outline.

Outline Shorthand

Para acortar el codigo, es posible especificar todas las propiedades de outline en una sola: outline-width outline-style outline-color.

Outline Offset

La propiedad outline-offset agrega espacio entre un outline y el borde/edge de un elemento. El espacio entre un elemento y su outline es transparente.
